3.3.0 Important Enhancements/Notes (4/11/2022)

On Use Effects

Where is my big update? We are currently trying to implement as many special abilities as possible. This consumes a lot of time but will result in great automation improvements. Compendium 2 has a huge coverage for effects and on use effects and gods of aventuria will soon too. Among a lot of other things those effects can now create a resistance roll check (e.g. don't get stunned while falling down). Not long ago Magic of Aventuria I received a similar update, so make sure to use the migration to update your special abilities!

Tokenhotbar v2

The token hotbar has been improved. You can now control it's direction and button size in the system settings and drag it to another location of your preference. Right click on the drag button changes the layout. Scrolling on the drag button changes the size. Moreover it does now contain additional buttons for "on-use-effects", system effects and active effects. You can increase the level of effects with a left click and remove or decrease the level with a right click. In addition you can add new system effects quickly via the add button. This also works if you selected several tokens at the same time.

Enhancements

  • Weapons now have settings for crit and botch rolls
  • No Target selected for attack rolls is now more obvious
  • Skill roll now shows selected targets
  • Targets can now be removed on roll dialogues
  • There is now a warning in the roll dialogue if you selected multiple targets
  • Skill rolls now have an additional "opposed roll" button, to prevent people from making opposed rolls accidentally
  • Rangeweapons can now have magazines. You can bring your shotgun to Dere now (or use your Eisenwalder more easily).
  • The assassination rule from the Compendium 2 has been added to the system. Find it on the attack dialogues.
  • Character Images from the character sheet and images in journals and the journal browser can now be dragged directly onto the canvas. This should faciliate the creation of theatre of mind scenes.

Bugfixes

  • Multiple tokens of the same unlinked actor should not get active effects applied via player interaction all at once anymore
  • Lucky hits in combat should be calculated properly again
  • Sight modifier for total darkness and attacks is working again
  • GM Arbitrariness now also works properly for damage rolls
