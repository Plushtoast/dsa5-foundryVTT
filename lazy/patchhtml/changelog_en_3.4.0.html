3.4.0 Important Enhancements/Notes (5/23/2022)

TDE Initiative Tracker

TDE now includes its own combattracker. It opens by default on the start of each combat and shows the next five turns. You can access the character sheet, target the tokens and everything else similar to the default combattracker. Moreover it allows to track actors which are not using their action immediately but instead are waiting. It can be disabled via the settings on your disposal.

Gods of Aventuria On-use effects

Gods of Aventuria I has been updated with a lot of on-use effects and active effects. Don't forget to migrate to gain access to the improvements.

Merchant V2

The merchant has been extended with a lot of new features:

  • Each item can now have a custom price
  • Items can be dropped via drag & drop to the current scene on the ground, resulting in a lootable token
  • Loot tokens can self delete once emptied
  • An optional chat message can be enabled via the settings so every transaction gets logged as chat message.
  • Creatures and Characters can now use the merchant sheet too (also allowing player trading)
  • Trading partners can be changed via drag and drop of another actor on the trading window (e.g. if the players own more than one actor)
  • Items can be excluded from the shop window
  • The header of icons now have subfunctions (lock category, increase amount of all items in the category...)
  • Random gear now has a field for quantity too
  • The core module now contains two macro to transform any token immediately into a lootsheet (so defeated creatures can be looted). The master's workshop extends this functionality with more details.

VTToms landing page

VTTom gifted his landing page to the system. Available in the core module.

Enhancements

  • Skill rolls can now be rolled as routine
  • The GM can now set targets for players. See the keybindings to set a custom hotkey or use it via the bar on the left side.
  • The reaction window now also shows the number of defenses.
  • Creatures with regeneration will have the regeneration included automatically with their next respective module update.
  • If the dere positioning system is enabled, doors can only be opened by nearby tokens.
  • The cumulative check is now closer to the rules. Critical rolls remove negative modifiers and rolls can be edited (e.g. via fatepoints)
  • Light presets from the light macro can now be added to the scene via drag & drop
  • The roll dialog should now show options for miracles, if your religion is configured properly
  • The tooltip for the roll target on d20 rolls is now hidden, if the user is not an owner of the rolling actor
  • Rabenkrieg 2 received an additional map - Das Kriegslager (v3.4.0). You can use it in the battle for trus or as encampment between scenes.
  • Rabenkrieg 2 now also has the actor "Daria"
  • The master's workshop can now show a list of example names via button and chatcommand (e.g. for character creation) and received some minor improvements (including loot transformation support, v3.4.0)

Bugfixes

  • The culture wizard does not add all languages as mother tongue anymore.
  • Crit & botch for unarmed attacks should work again
  • Some on use effects fixed in core module (v3.4.0)
