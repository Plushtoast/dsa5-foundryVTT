Grimorum Cantiones Update

All spell, ritual and cantrip changes from the "Grimorum Cantiones" book have been included in all existing modules for free. For the german version (no translation available) this also means additional information in the description like gestures and the effect of "Reversalis". For any language attributes have been adjusted, so spells might work a little bit different now. Make sure to migrate your spell casters.
Addtional spells only available in the Grimorum have been added to the module "Studierstube der Gildenmagie". There will be no separate "Grimorum" module.
German Users can have a short explanation here: VTToms Youtube channel

Yendor's UI

Yendor didn't like the way we styled the journals. So he made his own style available here: Yendor's UI
Maybe you like his style too.

The module provides an alternative DSA design for journals with new possibilities to individually design the content of the journals. Included are the well-known "For reading aloud or retelling" or also the pergment boxes from the DSA5 publications or also picture masks for a more beautiful presentation of pictures. More features are planned for the future.

(I am afraid the TDE 4.3.0 changes already break this module, but I bet he'll fix it soon.)
