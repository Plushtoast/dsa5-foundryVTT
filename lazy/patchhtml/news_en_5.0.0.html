Foundry DSA News (19.5.2023)

Foundry 11

Foundry has now been released in version 11 and DSA is already ready for it. Besides some new features, however, there are some minor problems, as with every new major version jump:

  • Foundry has changed how it handles status effects. As a result, leftover statuses on characters may appear twice or no longer function properly. These may have to be removed manually.
  • Foundry now handles unlinked tokens differently (tokens whose actor is independent of the source). This can cause scenes to get a bit buggy or large tokens before migrating to V11, removing and re-placing these should improve performance.
