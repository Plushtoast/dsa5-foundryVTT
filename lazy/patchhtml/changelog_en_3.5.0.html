3.5.0 Important Enhancements/Notes (7/11/2022)

Enhancements

  • The display of temporary regeneration has been changed a bit
  • Encumbrance no longer affects ranged combat (but this can be re-enabled as a house rule system setting)
  • Group checks samples can now also include different talents
  • Language can now be "forced" for all clients. Players whose language is different from the world language will receive a warning. Different language settings can cause some functions to not work properly.
  • The button for transferring effects is now locked for a short time to prevent multiple triggering.
  • Confirmation roll of 1W20 checks can now be modified via manual rolls or SL arbitrariness.
  • NPCs may now also roll regeneration.
  • DSA combat tracker now focuses on current turn (system setting, already via previous hotfix).
  • SL arbitrariness or manual rolls now select the complete roll on the first click into the input field (before: +- 1 function was triggered)
  • Hit points of spells can now also be modified via SL arbitrariness and manual throws.
  • Dialogs for checks now show the expected final modified skill value (already via previous hotfix).
  • Full text search for the library can now be disabled to save resources (requires reload)
  • The light templates in the basic rulebook have been improved a bit
  • The master's workshop now has a chatcommand for random names

Bugfixes

  • The roman level of special skills is now displayed up to level 10
  • The battle tracker now also shows token images with '(' in the name correctly
